Ход работы

Проектирование приложения: построена use-case диаграмма приложения PlantPal, построена архитектура экранов. Диаграммы находятся в папке: практическая работа 1.
Описание проекта: приложение должно помогать пользователям:

  вести каталог растений;
  
  сохранять избранные растения;
  
  просматривать карточки растений с деталями;
  
  получать информацию о частоте полива;
  
  добавлять заметки к растениям;
  
  распознавать растения по фото.
  
Пользователь может работать как гость (ограниченный функционал), так и авторизованный пользователь (полный функционал). Гость может просматривать список растений, искать растение, открывать карточку и получать частоту полива. Авторизованный пользователь дополнительно получает возможность сохранять растения в избранное, добавлять заметки и распознавать растение по фото. Внешний сервис используется для получения частоты полива. Приложение состоит из следующих экранов: список растений – общий каталог, возможность поиска. Карточка растения – отображает информацию о растении и частоту полива. Избранное – список растений, выбранных пользователем. Заметки – записи, связанные с растением. Экран авторизации/регистрации – для входа в систему.

В рамках первой практической работы было сделано:

Создание проекта:

  Создан Android-проект в Android Studio.
  Реализована структура по слоям:
  
    presentation — активити и UI;
    
    domain — модели, use-case классы;
    
    data — репозиторий для работы с данными.
